# Millonario Moderno

Juego web inspirado en “Quién quiere ser millonario”, con una interfaz más moderna.

## Archivos incluidos

- `index.html` — Página principal
- `index.css` — Estilos
- `index.js` — Lógica del juego
- `intro.mp4` — Video de introducción (debes agregarlo tú)

## Cómo publicarlo en GitHub Pages

1. Sube todos los archivos a tu repositorio.
2. Ve a **Settings → Pages**.
3. Selecciona:
   - Source: **Deploy from branch**
   - Branch: **main**
   - Folder: **/** (root)
4. Guardar.

¡Listo! Tu página queda publicada.
